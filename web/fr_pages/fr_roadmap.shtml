Roadmap :
Dernière modification: 21/08/00.

Version 0.1
Développement (estimation) : environ 1 mois
Date de sortie (estimation) : fin septembre
* Pré requis : Site web

* Graphismes :
- Textures basiques pour la mer, le sol, avec des transitions
(sur fond transparent)
- Quelques arbres (sur fond transparent)
- 1 Château / base
- 1 Lord (avec ses animations ?)
- Divers: flèches, curseurs, drapeaux (pour les bâtiments et les Lords)

* Programmation :
- Editeur de cartes
- Les Lords peuvent bouger sur les cartes
- Les Lords peuvent entrer dans les chateaux
- Les chateaux peuvent être améliorés

Développement (estimation) : 3 mois

* Graphismes :
