-- Les unités --

On distingue quatre grands types de pouvoirs :
1) Militaire
2) Biotechno
3) Homme d'influence/politique/d'affaire
4) Informatique et électronique

Pour chacun de ces pouvoirs, il existe un type de Lord avec une base et des unités spécifiques. Pour un type de pouvoir, il existe 7 unités de base, pouvant être upgradées chacune une fois.

Détail des unités :
