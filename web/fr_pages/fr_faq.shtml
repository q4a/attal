La FAQ reprend toutes les questions que vous pouvez vous poser sur Attal (programmation, graphismes, campagnes, intégration des thèmes etc.) et y répond (du mieux possible).

  • Comment utiliser les thèmes dans Attal ?
    Pour changer du thème par défaut il vous suffit de rajouter l'option '-theme' au lancement du client, suivi du nom du thème. Il existe pour l'instant deux thèmes : 'medieval' et 'cyberpunk'
  • Pourquoi les graphismes du theme 'medieval' sont-ils si peu
    travaillés ?
    Les graphistes actuels d'Attal ont décidé de se focaliser sur un seul thème ('cyberpunk') qui leur tenait à coeur et qui occuppe déjà tout leur temps. Le thème 'medieval' est développé très rapidement pour l'instant en bricolant quelques images. Si vous êtes intéressé par ce thème et que vous avez quelques talents de graphiste, nous accepterons volontiers vos contributions.
  • Je n'arrive pas à compiler Attal. Que faut-il faire ?
    Lire le fichier INSTALL !! Si cela ne marche toujours pas, me contacter.
  • Peut-on (ou pourra-t-on) jouer à Attal sous Windows ?
    Les sources d'Attal sont écrites en C++ et sont donc portables. Elles utilisent cependant une bibliothèque : Qt. Cette dernière est disponible sous Unix et sous Windows. Le problème est que la version Windows est (pour l'instant) payante, même si le programme est sous GPL. Nous n'avons pas les moyens pour l'instant de nous offrir une licence Windows de cette bibliothèque, mais pourquoi pas plus tard...
