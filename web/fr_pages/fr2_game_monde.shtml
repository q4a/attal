-- Ambiance générale --

Le jeu se déroule dans les années 2070.
Economie capitaliste développée à outrance.
Quelques importantes Corporations ont main mise sur le pouvoir politique des pays les plus avancés.
La rentabilité étant le maître mot, des domaines tels que l'écologie ont complètement été abandonnés.

-- Les technologies informatiques / telecom --

Les technologies issues de l'informatique et des télécommunications sont omniprésentes.

-- Les drogues / stimulants --

Les drogues, médicaments et autres dopants sont d'une efficacité redoutable et sont courament utilisées (tant par les civils que par les militaires ou certaines milices). Ces drogues provoquent des effets secondaires variables, pouvant être catastrophiques pour le corps humain sur le long terme.
